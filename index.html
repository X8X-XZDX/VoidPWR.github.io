Void Linux for PowerPC/Power ISA (unofficial)

Void Linux ppc was a staging fork of the Void Linux distribution intended for the PowerPC/Power architecture. At this point the project is unmaintained.

The author has moved on to maintain Chimera Linux, which also supports the POWER architecture.
